# Lead Management System

## 📌 Overview

The Lead Management System is a full-stack web application developed to manage customer leads efficiently. It enables users to add, view, update, and delete lead information through a simple and user-friendly interface.

---

## 🚀 Features

- Add new leads
- View all leads
- Update lead status
- Delete leads
- User-friendly interface
- Form validation
- PostgreSQL database integration

---

## 🛠️ Technologies Used

- React.js
- Node.js
- Express.js
- PostgreSQL
- JavaScript
- HTML5
- CSS3

## ⚙️ Installation

### Clone the repository

bash
git clone https://github.com/pallapudharani19-spec/LeadManagmentSystem.git


### Backend Setup

bash
cd backend
npm install
node server.js


### Frontend Setup

bash
cd frontend
npm install
npm run dev
