# 🚀 Lead Management System

A full-stack Lead Management System built using:

- React (Frontend)
- Node.js + Express (Backend)
- PostgreSQL (Neon DB)
- Deployed on Vercel + Render

---

## 🔗 Live Links

👉 Frontend (Vercel):  
https://lead-managment-system-737jrepn8-dharani1905.vercel.app

👉 Backend (Render API):  
https://leadmanagementsystem-wpdy.onrender.com

---

## ✨ Features

- Add new leads
- View all leads
- Update lead status
- Delete leads
- Persistent database storage (Neon PostgreSQL)
